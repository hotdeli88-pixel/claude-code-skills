# AIEP LCMS 저작도구 콘텐츠 자동 생성

## 개요

AIEP LCMS(Learning Content Management System)에서 제공하는 8가지 저작도구에 맞는 학습 콘텐츠를 자동 생성하는 스킬입니다.

**LCMS 접근 경로**: 수업콘텐츠 > 수업콘텐츠 관리 > 저작도구

### 지원 저작도구 8종

| # | 저작도구 | 코드명 | 설명 |
|---|---------|--------|------|
| 1 | 인터렉티브 비디오 | `interactive_video` | YouTube/영상 URL + 타임스탬프별 퀴즈 |
| 2 | 인터렉티브 슬라이드 | `interactive_slide` | 슬라이드별 텍스트+이미지+퀴즈 |
| 3 | 빈칸 채우기 | `fill_blanks` | 핵심 개념 문장에서 주요 단어를 빈칸으로 |
| 4 | 드래그 텍스트 | `drag_text` | 문장에서 단어를 드래그하여 올바른 위치에 배치 |
| 5 | 단어 선택 | `word_select` | 문장에서 올바른/틀린 단어를 클릭하여 선택 |
| 6 | 객관식 문제(복수형) | `mc_multi` | 정답이 2개 이상인 객관식 (체크박스) |
| 7 | 객관식 문제(단일형) | `mc_single` | 정답이 1개인 객관식 (라디오버튼) |
| 8 | O/X 퀴즈 | `ox_quiz` | 참/거짓 판단 문제 |

---

## 입력 요구사항

사용자에게 다음 정보를 확인합니다. 누락된 항목은 AskUserQuestion으로 질문합니다.

| 항목 | 필수 | 예시 | 비고 |
|------|------|------|------|
| 교과목 | O | 수학, 국어, 영어, 과학 등 | |
| 학년 | O | 중1, 중2, 중3 등 | |
| 단원명 또는 주제 | O | "유리수와 순환소수", "일차함수" | |
| 저작도구 타입 | O | 전체 / 선택 (쉼표 구분) | 기본값: 전체 |
| 난이도 | - | 기본 / 심화 | 기본값: 기본 |

**입력 예시**: `/aiep-authoring 수학 중2 일차함수 fill_blanks,mc_single 기본`

인자가 부족하면 대화로 확인합니다.

## 워크플로우

```
Step 1: 사용자에게 교과/학년/단원 확인
        - 인자 파싱, 누락 시 AskUserQuestion
Step 2: 생성할 저작도구 타입 결정
        - "전체"면 8종 모두, 아니면 선택된 것만
Step 3: 교과 내용 분석
        - 교육과정 성취기준 기반으로 핵심 개념, 용어, 공식 정리
        - 단원의 학습 목표와 평가 요소 파악
Step 4: 선택된 타입별 콘텐츠 JSON 생성
        - 아래 "저작도구별 생성 규칙"에 따라 JSON 구조 생성
        - 작업 디렉터리에 authoring_generator.py 스크립트 작성/실행
Step 5: 생성 결과를 파일로 저장
        - {단원명}_authoring_content.json (통합)
        - 타입별 개별 JSON 파일
Step 6: 결과를 사용자에게 표시
        - 각 도구별 미리보기 (문항 수, 샘플)
Step 7: (선택) LCMS 등록 안내
```

## 저작도구별 생성 규칙

### 1. 빈칸 채우기 (fill_blanks)

핵심 개념 문장에서 주요 키워드를 빈칸으로 만듭니다.

**생성 규칙**:
- 문항 수: 5~10개
- 각 문장에서 핵심 키워드 1~2개를 `*단어*` 형태로 빈칸 표시
- 빈칸은 정의, 공식, 성질의 핵심 용어에 설정
- hint는 빈칸 단어를 직접 노출하지 않되 추론 가능하게

**출력 JSON 구조**:
```json
{
  "type": "fill_blanks",
  "title": "{단원명} - 빈칸 채우기",
  "subject": "{교과}",
  "grade": "{학년}",
  "questions": [
    {
      "text": "분모를 *10*의 거듭제곱으로 나타낼 수 있는 분수를 *유한소수*라 한다.",
      "blanks": ["10", "유한소수"],
      "hint": "소수점 아래 끝이 있는 소수"
    }
  ]
}
```

---

### 2. 드래그 텍스트 (drag_text)

문장의 빈칸에 올바른 단어를 드래그하여 배치하는 문제입니다.

**생성 규칙**:
- 문항 수: 5~8개
- `[___]`로 빈칸 위치 표시
- drag_words에 드래그 가능한 단어 목록 (정답 + 오답 포함 가능)
- correct_order로 각 빈칸에 들어갈 drag_words의 인덱스 지정

**출력 JSON 구조**:
```json
{
  "type": "drag_text",
  "title": "{단원명} - 드래그 텍스트",
  "subject": "{교과}",
  "grade": "{학년}",
  "questions": [
    {
      "text": "y = [___]x + [___] 에서 기울기는 [___]이고 y절편은 [___]이다.",
      "drag_words": ["2", "3", "2", "3"],
      "correct_order": [0, 1, 0, 1]
    }
  ]
}
```

---

### 3. 단어 선택 (word_select)

문장에서 올바른 또는 틀린 단어를 클릭하여 선택하는 문제입니다.

**생성 규칙**:
- 문항 수: 5~8개
- 문장 내 선택 가능한 단어를 select_words로 지정
- correct: true (올바른 단어) / false (틀린 단어)
- 문장 자체가 참이면 핵심 단어가 correct=true, 거짓 문장이면 틀린 부분이 correct=false

**출력 JSON 구조**:
```json
{
  "type": "word_select",
  "title": "{단원명} - 단어 선택",
  "subject": "{교과}",
  "grade": "{학년}",
  "questions": [
    {
      "text": "주사위를 던질 때 짝수가 나올 확률은 1/2이다.",
      "select_words": [{"word": "1/2", "correct": true}]
    },
    {
      "text": "동전을 던질 때 앞면이 나올 확률은 1/3이다.",
      "select_words": [{"word": "1/3", "correct": false, "correction": "1/2"}]
    }
  ]
}
```

---

### 4. 객관식 문제 - 단일형 (mc_single)

정답이 1개인 객관식 문제입니다.

**생성 규칙**:
- 문항 수: 5~10개
- 보기 4~5개 (정답 1개 + 오답 3~4개)
- answer는 정답 보기의 인덱스 (0-based)
- 오답은 그럴듯하되 명확히 구별 가능하게 (흔한 오개념 활용)
- explanation에 풀이 과정 포함

**출력 JSON 구조**:
```json
{
  "type": "mc_single",
  "title": "{단원명} - 객관식(단일)",
  "subject": "{교과}",
  "grade": "{학년}",
  "questions": [
    {
      "question": "직각삼각형에서 빗변의 길이가 5, 한 변의 길이가 3일 때 나머지 변의 길이는?",
      "options": ["2", "3", "4", "6"],
      "answer": 2,
      "explanation": "5^2 = 3^2 + x^2, x^2 = 25 - 9 = 16, x = 4"
    }
  ]
}
```

---

### 5. 객관식 문제 - 복수형 (mc_multi)

정답이 2개 이상인 객관식 문제입니다.

**생성 규칙**:
- 문항 수: 3~7개
- 보기 4~5개, 정답 2개 이상
- answers는 정답 보기의 인덱스 배열 (0-based)
- "모두 고르시오" 형태의 질문

**출력 JSON 구조**:
```json
{
  "type": "mc_multi",
  "title": "{단원명} - 객관식(복수)",
  "subject": "{교과}",
  "grade": "{학년}",
  "questions": [
    {
      "question": "다음 중 무리수인 것을 모두 고르시오.",
      "options": ["sqrt(2)", "0.333...", "pi", "1/3"],
      "answers": [0, 2],
      "explanation": "sqrt(2)와 pi는 순환하지 않는 무한소수이므로 무리수이다."
    }
  ]
}
```

---

### 6. O/X 퀴즈 (ox_quiz)

참/거짓을 판단하는 문제입니다.

**생성 규칙**:
- 문항 수: 5~10개
- 참(true)과 거짓(false)의 비율을 대략 1:1로 유지
- 거짓 문항은 흔한 오개념을 활용
- explanation에 참/거짓 이유를 간결하게 설명

**출력 JSON 구조**:
```json
{
  "type": "ox_quiz",
  "title": "{단원명} - O/X 퀴즈",
  "subject": "{교과}",
  "grade": "{학년}",
  "questions": [
    {
      "statement": "모든 정사각형은 직사각형이다.",
      "answer": true,
      "explanation": "정사각형은 네 각이 모두 직각이므로 직사각형의 조건을 만족한다."
    },
    {
      "statement": "두 삼각형의 넓이가 같으면 합동이다.",
      "answer": false,
      "explanation": "넓이가 같아도 변의 길이와 각이 다를 수 있으므로 합동이 아닐 수 있다."
    }
  ]
}
```

### 7. 인터렉티브 비디오 (interactive_video)

영상 재생 중 특정 시점에 퀴즈가 등장하는 형태입니다.

**생성 규칙**:
- video_url은 빈 문자열로 생성 (사용자가 직접 입력)
- 타임스탬프별 인터랙션 3~7개
- 각 인터랙션은 mc_single, fill_blanks, ox_quiz 중 하나
- 영상 흐름에 맞게 개념 도입 -> 예제 -> 심화 순서로 배치

**출력 JSON 구조**:
```json
{
  "type": "interactive_video",
  "title": "{단원명} - 인터렉티브 비디오",
  "subject": "{교과}",
  "grade": "{학년}",
  "video_url": "",
  "video_description": "영상 URL을 입력해주세요.",
  "interactions": [
    {
      "timestamp": "1:30",
      "label": "개념 확인",
      "type": "mc_single",
      "question": "일차함수 y = ax + b에서 a가 의미하는 것은?",
      "options": ["x절편", "y절편", "기울기", "상수"],
      "answer": 2
    },
    {
      "timestamp": "3:00",
      "label": "핵심 용어",
      "type": "fill_blanks",
      "text": "일차함수의 그래프에서 x축과 만나는 점의 좌표를 *x절편*이라 한다.",
      "blanks": ["x절편"]
    },
    {
      "timestamp": "5:00",
      "label": "참거짓 확인",
      "type": "ox_quiz",
      "statement": "기울기가 음수인 일차함수의 그래프는 오른쪽 아래로 향한다.",
      "answer": true
    }
  ]
}
```

### 8. 인터렉티브 슬라이드 (interactive_slide)

슬라이드별로 학습 내용과 퀴즈를 함께 제공합니다.

**생성 규칙**:
- 슬라이드 5~10개
- 구성: 개념 소개 (2~3장) -> 예제 (2~3장) -> 연습 (2~3장) -> 정리 (1장)
- 각 슬라이드에 content(학습 내용)와 interaction(퀴즈) 포함
- interaction은 mc_single, fill_blanks, ox_quiz 중 택1

**출력 JSON 구조**:
```json
{
  "type": "interactive_slide",
  "title": "{단원명} - 인터렉티브 슬라이드",
  "subject": "{교과}",
  "grade": "{학년}",
  "slides": [
    {
      "slide_number": 1,
      "title": "개념 소개: 일차함수란?",
      "content": "y = ax + b (a != 0) 형태의 함수를 일차함수라 합니다.\n\n- a: 기울기 (x가 1 증가할 때 y의 변화량)\n- b: y절편 (그래프가 y축과 만나는 점의 y좌표)",
      "interaction": {
        "type": "mc_single",
        "question": "다음 중 일차함수인 것은?",
        "options": ["y = 3", "y = x^2 + 1", "y = 2x - 5", "y = 1/x"],
        "answer": 2,
        "explanation": "y = 2x - 5는 y = ax + b 형태(a=2, b=-5)이므로 일차함수이다."
      }
    }
  ]
}
```

## 콘텐츠 생성 품질 기준

### 교육과정 정합성
- 해당 학년 교육과정 성취기준에 부합하는 내용만 출제
- 상위 학년 개념을 사용하지 않음
- 교과서에서 사용하는 용어와 표기법 준수

### 문항 품질
- 오개념을 유발하지 않는 정확한 내용
- 객관식 오답은 그럴듯하되 명확히 틀린 것 (흔한 실수 유형 활용)
- 풀이(explanation)는 학생이 이해할 수 있는 수준으로 작성
- 수학 기호는 텍스트로 표현 (sqrt, ^, pi 등)

### 난이도 조절
- **기본**: 교과서 예제/확인문제 수준, 개념 정의 위주
- **심화**: 응용/서술형 수준, 여러 개념을 결합한 문제

### 문항 수 가이드
| 저작도구 | 기본 | 심화 |
|---------|------|------|
| fill_blanks | 5~7문항 | 8~10문항 |
| drag_text | 5~6문항 | 7~8문항 |
| word_select | 5~6문항 | 7~8문항 |
| mc_single | 5~7문항 | 8~10문항 |
| mc_multi | 3~5문항 | 5~7문항 |
| ox_quiz | 5~7문항 | 8~10문항 |
| interactive_video | 인터랙션 3~5개 | 5~7개 |
| interactive_slide | 슬라이드 5~7개 | 8~10개 |

## LCMS 수동 등록 안내

생성된 JSON 콘텐츠를 LCMS에 등록하는 절차:

1. AIEP LCMS에 로그인
2. **수업콘텐츠** > **수업콘텐츠 관리** > **저작도구** 메뉴 진입
3. **등록** 버튼 클릭
4. 저작도구 드롭다운에서 해당 타입 선택 (예: "빈칸 채우기")
5. 교과, 학년, 단원 정보 입력
6. 생성된 콘텐츠를 편집기에 입력:
   - 텍스트 기반 도구: 각 문항을 에디터에 직접 입력
   - 인터렉티브 도구: 슬라이드/타임스탬프별로 구성
7. 미리보기로 확인 후 **저장** 클릭

## 주의사항

- 인터렉티브 비디오의 video_url은 빈 문자열로 생성됩니다. 사용자가 직접 YouTube URL 등을 입력해야 합니다.
- JSON 출력은 UTF-8 인코딩으로 저장합니다.
- 수학 수식은 LaTeX가 아닌 텍스트 표기를 사용합니다 (LCMS 에디터 호환).
- 생성된 콘텐츠는 반드시 교사가 검토 후 사용해야 합니다.
