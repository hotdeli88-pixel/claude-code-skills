# 전북형 CBCI 단원설계 평가 (cbci-unit-eval)

## 개요

전북형 개념기반 탐구학습(CBCI) 단원설계 문서를 체계적으로 분석하여, 설계 절차 8단계 이행도·탐구 질문 체계·탐구 기법(ATL) 매핑·3차원 평가 체계·소주제 간 연결성을 평가하고, 100점 만점 보고서(MD)를 출력한다.

## 사용법

```
/cbci-unit-eval <단원설계 파일 경로>
/cbci-unit-eval ./이차방정식_단원설계.hwpx
/cbci-unit-eval ./단원설계.docx ./보고서_출력.md
/cbci-unit-eval            # 인자 없으면 사용자에게 경로 질문
```

### 지원 파일 형식

| 형식 | 확장자 | 처리 방법 |
|------|--------|----------|
| HWPX | `.hwpx` | hwpx 스킬로 읽기 또는 ZIP 해제 후 XML 파싱 |
| DOCX | `.docx` | Read 도구로 직접 읽기 |
| PDF | `.pdf` | Read 도구로 직접 읽기 (이미지 기반이면 페이지별 Read) |
| Markdown | `.md` | Read 도구로 직접 읽기 |
| 텍스트 | `.txt` | Read 도구로 직접 읽기 |

## 평가 체계 (100점)

| 영역 | 배점 | 세부 항목 |
|------|------|----------|
| 1. 설계 절차 8단계 이행 | 40 | 단원명(5), 성취기준(5), 핵심개념(5), 핵심기능(5), 학생상·핵심가치(5), 단원관련개념(5), 일반화(5), 학습모형(5) |
| 2. 탐구 질문 체계 | 15 | 단원핵심질문(5), 소주제별 사실·개념·논쟁적 질문(10) |
| 3. 탐구 기법(ATL) 적용 | 20 | 7단계 매핑 완결성(10), 기법 적합성(10) |
| 4. 3차원 평가 체계 | 15 | 차원별 정합성(10), 평가방법 적합성(5) |
| 5. 소주제 간 연결성·전이 | 10 | 소주제 간 브리지(5), 핵심개념 추적(5) |
| **합계** | **100** | |

### Phase 1: 문서 읽기 및 구조 추출

1. **파일 형식에 따른 읽기**:
   - HWPX: ZIP 해제 → XML 파싱, 또는 Read 도구로 직접 읽기

2. **구조화된 정보 추출** — 다음 항목을 식별·추출한다:

   ```
   ┌─ 기본 정보
   │  ├─ 학년/교과
   │  ├─ 단원명
   │  ├─ 전체 차시 / 소주제 수
   │  ├─ 운영 유형
   │  └─ 학습 모형
   │
   ├─ 핵심 구성요소
   │  ├─ 학생상
   │  ├─ 핵심 가치 (나열)
   │  ├─ 핵심 개념 (주/보조 구분)
   │  ├─ 핵심 기능 (나열)
   │  ├─ 단원 관련 개념 (나열)
   │  └─ 성취기준 (코드 + 내용)
   │
   ├─ 일반화 체계
   │  ├─ 단원 일반화
   │  └─ 소주제별 일반화 (각 소주제)
   │
   ├─ 탐구 질문
   │  ├─ 단원 핵심 질문
   │  └─ 소주제별 질문 (사실적/개념적/논쟁적 × 각 소주제)
   │
   ├─ 탐구 기법 매핑
   │  └─ 소주제별 × 7단계별 기법 목록
   │     (관계맺기/집중하기/조사하기/조직및정리/일반화/전이/성찰)

   │  └─ 각 차시: 탐구단계, 핵심활동, 탐구기법, PYP발문

   └─ 3차원 평가
      ├─ 1차원 (차시, 내용, 방법)
      ├─ 2차원 (차시, 내용, 방법)
      └─ 3차원 (차시, 내용, 방법)
   ```

3. 추출이 불완전한 항목은 `[미확인]`으로 표시하고 평가 시 감점 근거로 활용한다.

### Phase 2: 영역별 평가

각 영역을 eval_criteria.md의 채점 기준에 따라 평가한다. 아래는 영역별 평가 절차이다.

#### 2-1. 설계 절차 8단계 이행 (40점)

각 단계별 5점 만점으로 평가한다.

| 단계 | 평가 관점 | 5점 기준 |
|------|----------|---------|
| 1단계: 단원명 | 핵심개념 반영, 흥미 유발 | 핵심개념이 단원명에 명시적으로 드러나며 학습자 탐구 동기를 유발 |
| 2단계: 성취기준 | 교육과정 정합성, 포괄성 | 해당 교육과정 성취기준을 정확히 선정하고 단원 범위를 포괄 |
| 3단계: 핵심개념 | 26개 중 적합한 선정, 주/보조 구분 | 26개 핵심개념 중 교과·단원 특성에 맞는 2~3개 선정, 주/보조 위계 명확 |
| 4단계: 핵심기능 | 9개 중 적합한 선정, 소주제별 차별화 | 9개 핵심기능 중 단원 목표에 부합하는 3~5개 선정, 소주제별 중점 기능 차별화 |
| 5단계: 학생상·핵심가치 | 5대 학생상·13개 핵심가치와의 정합성 | 5대 학생상 중 2개 이상과 직접 연결, 핵심가치가 교과 특성에 부합 |
| 6단계: 단원관련개념 | 포괄성, 소주제 대응, 위계 | 핵심개념에서 파생되는 관련 개념이 풍부하고 소주제 순서와 위계 일치 |
| 7단계: 일반화 | 4대 원칙 충족, 체인 정합성 | 단원·소주제 일반화가 4대 원칙(구조·보편·복잡·완결)을 모두 충족하고 체인 구조가 논리적 |
| 8단계: 학습모형 | 모형 선택 근거, 소주제별 적합성 | 소주제의 지식 유형에 맞는 모형 선택, 모형 간 상보성 구현 |

**채점 세부 기준 (각 단계 공통):**
- 5점: 기준을 모두 충족하며 모범적
- 4~4.5점: 기준을 대부분 충족하나 경미한 보완 필요
- 3~3.5점: 기준을 부분적으로 충족, 구체적 개선 필요
- 2~2.5점: 기준 미충족 항목이 다수, 상당한 개선 필요
- 1~1.5점: 해당 단계가 거의 설계되지 않음

#### 2-2. 탐구 질문 체계 (15점)

| 세부 항목 | 배점 | 5점 기준 |
|----------|------|---------|
| 단원 핵심 질문 | 5 | 개방적·포괄적이며 12차시 전체를 관통, 개념적+논쟁적 요소 포함 |
| 소주제별 질문 | 10 | 사실적·개념적·논쟁적 각 3개씩 균형 배치(소주제당 9개), 계층적 구조 명확 |

**질문 체계 채점 세부 기준:**
- 사실적 질문: 구체적 수치·사례 포함, 관찰 가능한 사실 기반
- 개념적 질문: '왜(why)', '어떻게(how)' 중심, 원리 탐구 유도
- 논쟁적 질문: 정답 없음, 다양한 관점, 비판적 사고 자극
- 질문 수 확인: 소주제별 사3+개3+논3 = 9개 × 소주제 수 = 총 질문 수
- 질문 누락 시 감점: 소주제당 질문이 9개 미만이면 비례 감점

#### 2-3. 탐구 기법(ATL) 적용 (20점)

| 세부 항목 | 배점 | 평가 기준 |
|----------|------|----------|
| 7단계 매핑 완결성 | 10 | 7단계(관계맺기~성찰) 각 단계에 1개 이상 기법이 매핑됨 |
| 기법 적합성 | 10 | 각 기법이 해당 탐구 단계에 적합하게 배치됨 |

**7단계 매핑 검증 체크리스트 (소주제별 각각 확인):**

| 탐구 단계 | 적합한 기법 유형 | 누락 시 감점 |
|----------|----------------|-------------|
| 관계맺기 | 의견기반전략, 경험기반전략, 토론기반전략, 시뮬레이션, 실험놀이 | -1.5 |
| 집중하기 | 분류전략, 명명전략(프레이어 모델), 순위전략, 다이아몬드 랭킹 | -1.5 |
| 조사하기 | 체험, 동료멘토링, 시뮬레이션, 출판자료, 테크놀로지 활용 | -1.5 |
| 조직 및 정리 | 비교조직자, 과정조직자, 요약조직자, 표상전략, T차트, 그래픽조직자 | -1.5 |
| 일반화하기 | 일반화 명료화, 일반화 정당화, 일반화 공유 | -1.5 |
| 전이하기 | 근전이 과제, 원전이 과제, 새로운 맥락 적용 | -1.0 |
| 성찰하기 | 3-2-1 성찰, 학습저널, 자기평가, 메타인지 활동 | -1.0 |

**기법 적합성 검증:**
- 각 기법이 올바른 탐구 단계에 배치되었는지 확인
- 기법이 단계 귀속이 불분명하면 감점 (예: 시뮬레이션이 관계맺기인지 조사하기인지 불명확)
- 소주제의 탐구 모형(발견/원리/과정)에 적합한 기법인지 확인

#### 2-4. 3차원 평가 체계 (15점)

| 세부 항목 | 배점 | 평가 기준 |
|----------|------|----------|
| 차원별 정합성 | 10 | 1차원(이해)→2차원(적용·추론)→3차원(종합·전이)의 단계적 심화 |
| 평가방법 적합성 | 5 | 각 차원에 적합한 평가 방법 선택 |

**3차원 평가 검증:**

| 차원 | 기대 수준 | 적합한 평가 방법 |
|------|----------|----------------|
| 1차원 | 사실적 지식·개념 이해 확인 | 서답형, 선택형, 구술 |
| 2차원 | 개념의 맥락적 적용, 추론 | 서술형, 수행과제 |
| 3차원 | 종합적 사고, 실제 문제 해결, 전이 | 보고서, 프로젝트, 발표, 포트폴리오 |

- 평가 차시가 소주제 마지막에 배치되는지 확인
- 3차원 평가에 실제 맥락(전이 과제)이 포함되는지 확인
- 각 차원 누락 시 해당 배점 전체 감점

#### 2-5. 소주제 간 연결성·전이 (10점)

| 세부 항목 | 배점 | 평가 기준 |
|----------|------|----------|
| 소주제 간 브리지 | 5 | 소주제 전환 시 연결 예고·브리지 활동 존재 |
| 핵심개념 추적 | 5 | 핵심 개념(주+보조)이 12차시 전체에 걸쳐 추적 가능 |

## 평가 원칙

1. **근거 기반 평가**: 모든 점수에 단원설계 문서에서 인용한 구체적 근거를 제시한다.
2. **전북형 용어 사용**: Wathall 원서 용어가 아닌 전북형 CBCI 워크북(2.pdf)의 한국어 용어를 사용한다.
3. **건설적 피드백**: 감점 항목에는 반드시 구체적이고 실행 가능한 개선 제안을 함께 제시한다.
4. **누락 항목 명시**: 문서에서 확인되지 않는 항목은 `[미확인]`으로 표시하고 감점 근거를 명시한다.
5. **정량+정성 균형**: 점수(정량)와 분석(정성)을 모두 포함하여 평가의 신뢰성을 높인다.
6. **우수 사례 인용**: 특별히 잘 설계된 부분은 구체적으로 인용하여 강점을 부각한다.
