# 전북형 개념기반 탐구학습 설계 — 프레임워크 레퍼런스

## 1. 세 가지 학습모형

### (1) 발견 중심형 (귀납적)
- **흐름**: 사실 → 개념 → 일반화
- **탐구 단계**: 관계맺기 → 집중하기(개념 속성 파악) → 조사하기 → 조직 및 정리 → 일반화+성찰
- **선택 기준**: 학생이 구체적 사례에서 패턴을 발견하여 개념을 구성할 때
- **특징**: 일반화를 먼저 제시하지 않음. 교사는 안내자 역할

### (2) 원리 탐구형 (연역적)
- **흐름**: 일반화 → 개념 → 사실
- **탐구 단계**: 관계맺기 → 일반화 살펴보기 → 개념 탐색 및 정의 → 일반화 적용+평가
- **선택 기준**: 이미 알려진 원리/법칙을 다양한 맥락에 적용하며 깊이 이해할 때
- **특징**: 일반화를 먼저 제시하고 검증/적용

### (3) 과정 구성형 (절차적)
- **흐름**: 예시 → 기능 → 일반화
- **탐구 단계**: 관계맺기 → 기능/전략 분석 → 기능/전략 적용 → 일반화+평가
- **선택 기준**: 절차적 지식을 개념적 사고와 통합하여 기능 숙달 및 전이를 목표로 할 때
- **특징**: 수행을 통해 일반화에 도달

### 모형 선택 가이드

| 단원 특성 | 추천 모형 |
|----------|---------|
| 개념 발견이 핵심 (패턴, 분류, 규칙성) | 발견 중심형 |
| 원리/공식의 적용과 확장이 핵심 | 원리 탐구형 |
| 절차/알고리즘의 이해와 수행이 핵심 | 과정 구성형 |
| 혼합 (개념 발견 + 절차 적용) | 소주제별 다른 모형 선택 가능 |

## 2. 26개 핵심개념 (4범주)

### 구조와 관계 (9개)
원인과 결과, 시스템, 주기, 상호작용, **패턴**, **변화**, 연결, **관계**, **형태**

### 사회와 가치 (9개)
다양성, 갈등과 협력, 공감과 소통, 공동체, 문화, **관점**, 평등, 인권, 공정

### 사고와 표현 (5개)
**논리**, 창의성, 심미성, 시간과 공간, 자유

### 실천과 기능 (3개)
정보, **기능**, 책임

> 수학 교과에서 자주 사용: 패턴, 관계, 형태, 변화, 논리, 기능, 원인과 결과, 시스템

## 3. PYP 8개 핵심개념 (발문용)

| 개념 | 핵심 질문 패턴 |
|------|-------------|
| 형태(Form) | "이것은 어떤 형태/모양을 가지는가?" |
| 기능(Function) | "이것은 어떻게 작동/기능하는가?" |
| 원인(Causation) | "왜 이렇게 되는가? 원인은?" |
| 변화(Change) | "어떻게 변화하는가?" |
| 연관성(Connection) | "다른 것과 어떻게 연결되는가?" |
| 관점(Perspective) | "다른 관점에서 보면 어떤가?" |
| 책임(Responsibility) | "우리의 책임은 무엇인가?" |
| 성찰(Reflection) | "무엇을 배웠고 어떻게 달라졌는가?" |

> 각 차시에 2개씩, 8개 고르게 분포

## 4. 탐구기법 — 단계별 매핑 원칙

| 탐구 단계 | 핵심 탐구기법 | 역할 |
|----------|-------------|------|
| 관계맺기 | 의견기반전략, 경험기반전략, 토론기반전략 | 호기심 촉발, 인지적 갈등, 사전지식 활성화 |
| 집중하기 | 분류전략, 명명전략(프레이어 모델), 순위전략 | 개념 경계 형성, 예시/비예시 분류 |
| 조사하기 | 체험, 동료멘토링, 시뮬레이션, 출판자료 | 사례 축적, 패턴 관찰, 데이터 수집 |
| 조직+정리 | 비교조직자(교차비교차트), 과정조직자(흐름도), 요약조직자, 표상전략 | 패턴 간 관계 시각화, 일반화 준비 |
| 일반화 | 일반화 명료화/정당화/공유 | 학생 자신의 언어로 일반화 진술 구성 |
| 전이 | 근전이(유사 맥락), 원전이(새 영역) | 소주제 초반=근전이, 후반=원전이 |
| 성찰 | 3-2-1 전략, 학습저널 | 메타인지 촉진 (매 소주제 마지막 필수) |

## 5. 일반화 작성 원칙

### 4대 원칙
1. **"개념1 + 관계동사 + 개념2" 구조** — 핵심개념 간 관계를 담은 문장
2. **보편성** — 시간/문화 초월, 특정 사례에 한정되지 않음
3. **복잡성** — 단순 정의가 아닌 인과/조건-결과-확장의 다층 논리
4. **완결성** — 주어-서술어를 갖춘 완전한 문장

### 검증 체크리스트
- [ ] 소주제 일반화들이 단원 일반화의 각 절(clause)과 1:1 대응하는가?
- [ ] 소주제 순서가 단원 일반화의 논리 흐름과 일치하는가?
- [ ] 단원 일반화가 성취기준을 포괄하면서 전이 수준으로 확장하는가?
- [ ] 핵심개념이 소주제별 역할(주렌즈/보조렌즈)이 명확한가?
- [ ] 탐구질문이 사실/개념/논쟁 각 3개씩 균형 배치되었는가?

## 6. 평가 계획 (3차원)

| 차원 | 목표 | 전형적 방법 |
|------|------|-----------|
| 1차원 (이해) | 개념/사실 이해 확인 | 서답형, 선택형 |
| 2차원 (적용) | 개념적 이해의 맥락 적용, 추론 | 서술형, 수행과제 |
| 3차원 (전이) | 새로운 맥락으로 전이, 실행 | 보고서, 프로젝트, 포트폴리오 |

> 순환적: 모든 학습자가 궁극적으로 3차원에 도달하도록 설계

## 7. 5가지 학생상

1. 자기 주도적으로 학습하는 학생
2. 새로움을 제안하고 대안을 마련하는 학생
3. 더불어 소통하고 협력하는 학생
4. 생태적으로 감수성이 풍부한 학생
5. 문화적 소양을 갖춘 학생

## 8. 13가지 핵심 가치

인간존중, 책임과 의무, 평등과 정의, 존중과 배려, 사회적 참여,
비판적 사고와 문제해결, 도전과 성장, 공감과 소통, 다양성 존중,
자율성과 자기관리, 사고의 유연성, 생태감수성, 문화적 소양

## 9. 9가지 핵심 기능

설명하기, 분석하기, 비교하기, 적용하기, 종합하기,
평가하기, 의사소통하기, 문제해결하기, 창안하기
