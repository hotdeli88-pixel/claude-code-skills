# 전북형 CBCI 단원설계 평가 기준 상세

## 개요

이 문서는 전북형 개념기반 탐구학습(CBCI) 단원설계의 평가 기준을 정의한다.
평가 근거는 전북형 CBCI 워크북(2.pdf)과 탐구기법(종합) ppt.pdf이다.

| 점수 | 기준 |
|------|------|
| 5 | 단원·소주제 일반화 모두 4대 원칙 충족, 소주제→단원 체인 구조 논리적, 소주제 일반화가 단원 일반화의 절(clause)과 1:1 대응 |
| 4 | 4대 원칙 대부분 충족, 체인 구조 양호하나 일부 일반화의 추상성 부족 |
| 3 | 일반화가 제시되었으나 사실 수준에 머무르거나 체인 구조 불분명 |

**일반화 4대 원칙:**
1. "개념1 + 관계동사 + 개념2" 구조
2. 보편성 — 시간·문화 초월, 특정 사례 한정 아님
3. 복잡성 — 단순 정의가 아닌 인과/조건-결과의 다층 논리
4. 완결성 — 주어-서술어를 갖춘 완전한 문장

**일반화 체인 검증:**
- [ ] 소주제 일반화들이 단원 일반화의 각 절(clause)과 1:1 대응하는가?
- [ ] 소주제 순서가 단원 일반화의 논리 흐름과 일치하는가?
- [ ] 단원 일반화가 성취기준을 포괄하면서 전이 수준으로 확장하는가?

### 소주제별 탐구 질문 (10점)

| 점수 | 기준 |
|------|------|

| 7~8 | 대부분 균형, 일부 소주제에서 질문 수 부족 또는 유형 편중 |
| 5~6 | 질문이 제시되었으나 유형 구분 불분명하거나 수 부족 |

**질문별 품질 기준:**

- 개념적(개): '왜', '어떻게', '~인 까닭' 등 원리 탐구 유도, 개념 간 관계 탐색
- 논쟁적(논): 정답 없음, 다양한 관점 가능, 비판적 사고 자극, 열린 토론 촉진

## 3. 탐구 기법(ATL) 적용 (20점)

### 7단계 매핑 완결성 (10점)

각 소주제별로 7단계 모두에 기법이 매핑되어야 한다.

**7단계별 적합한 기법 유형:**

| 탐구 단계 | 전략 유형 | 대표 기법 |
|----------|----------|----------|
| 관계맺기 | 의견기반, 경험기반, 토론기반 | 네모통이 토론, 스펙트럼, 거미줄 프로토콜, 시뮬레이션, 실험놀이, See-Think-Wonder |
| 집중하기 | 분류, 명명, 순위 | 프레이어 모델, 개념 사분면, 스펙트럼 정렬, 다이아몬드 랭킹 |
| 조사하기 | 자료 수집, 체험 | 출판자료, 테크놀로지(GeoGebra/Desmos 등), 영상, 체험, 동료멘토링 |
| 조직 및 정리 | 비교조직자, 과정조직자, 요약조직자, 표상 | 교차비교차트, T차트, 밀고당기고잡기, 흐름 다이어그램, 누락된 조각, 시각적 메모, 그래픽 조직자, Thinking Map |
| 일반화하기 | 일반화 구성 | 일반화 명료화, 정당화, 공유, 핵심어 수정 |
| 전이하기 | 적용 | 근전이(유사 맥락), 원전이(새 영역), 수행평가 과제 |
| 성찰하기 | 메타인지 | 3-2-1 성찰, 학습저널, 자기평가, 성찰일지 |

**감점 규칙:**
- 소주제에서 특정 단계의 기법이 완전 누락: 해당 단계당 -1.0~1.5
- 모든 소주제에서 동일 단계 누락: 추가 -1.0
- 매핑은 있으나 기법명이 불분명: -0.5

### 기법 적합성 (10점)

| 점수 | 기준 |
|------|------|
| 9~10 | 모든 기법이 해당 탐구 단계에 적합, 모형별 특성 반영, 교과 맥락에 맞는 활용 |
| 7~8 | 대부분 적합, 일부 기법의 단계 귀속이 불분명 |
| 5~6 | 기법 배치가 부분적으로 적합, 단계 혼동 다수 |
| 3~4 | 기법이 나열되었으나 단계별 매핑이 불분명 |
| 1~2 | 탐구 기법이 거의 미제시 |
