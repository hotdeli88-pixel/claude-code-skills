# 교수학습-평가 운영 계획 JSON 데이터 형식

## JSON 스키마

```json
{
  "school_info": {
    "school_name": "OO중",
    "subject": "수학",
    "semester": "1학기",
    "year": "2026",
    "grades": [
      {
        "grade": 1,
        "classes": 6,
        "hours_per_week": 4,
        "textbook": "수학1(○○출판사)"
      }
    ]
  },
  "curriculum_tables": [
    {
      "grade": 1,
      "rows": [
        ["3월 1주 (3.3~3.6)", "4", "4", "I. 소인수분해", "[9수01-01] ...", "§ 평가요소...", "[수업]...[평가]...", ""],
        ...
      ]
    }
  ]
}
```

## 필드 설명

### school_info (필수)
| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| subject | string | O | 교과명 (수학, 국어, 영어 등) |
| semester | string | O | 학기 (1학기, 2학기) |
| year | string | - | 학년도 (기본 2026) |
| school_name | string | - | 학교명 (기본 OO중) |
| grades | array | O | 학년 정보 배열 |

### grades[] (학년 정보)
| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| grade | int | O | 학년 (1, 2, 3) |
| classes | int | - | 학급 수 |
| hours_per_week | int | - | 주당 시수 (기본 4) |
| textbook | string | - | 교과서명 |

### curriculum_tables[] (교수학습-평가 계획)
| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| grade | int | O | 해당 학년 |
| rows | array | O | 8열 데이터 행 배열 |

### rows[] (8열 데이터)
리스트 형식 또는 딕셔너리 형식 모두 지원.

#### 리스트 형식 (권장)
```
[시기, 시수, 누계, 단원명, 성취기준, 평가요소, 수업평가방법, 비고]
```

#### 딕셔너리 형식
```json
{
  "period": "3월 1주",
  "hours": "4",
  "cumulative": "4",
  "unit": "I. 소인수분해",
  "standards": "[9수01-01] ...",
  "eval_elements": "§ ...",
  "methods": "[수업]...[평가]...",
  "notes": ""
}
```

## 8열 컬럼 상세

| # | 컬럼명 | 키 | 설명 | 예시 |
|---|--------|-----|------|------|
| 0 | 시기 | period | 주차/날짜 | "3월 1주 (3.3~3.6)" |
| 1 | 시수 | hours | 해당 주 수업시수 | "4" |
| 2 | 누계 | cumulative | 누적 시수 | "4" |
| 3 | 단원명 | unit | 단원 번호+이름 | "I. 소인수분해" |
| 4 | 교육과정 성취기준 | standards | 성취기준 코드+내용 | "[9수01-01] ..." |
| 5 | 평가 요소 | eval_elements | 평가요소 및 평가유형 | "§ 소인수분해의 뜻..." |
| 6 | 수업·평가 방법 | methods | 수업방법+에듀테크+평가방법 | "[수업]\n§ 발문식..." |
| 7 | 비고 | notes | 수행평가 표시 등 | "수행평가1" |

## 성취기준 코드 형식

교과별 성취기준 코드 형식:
- 수학: `[9수01-01]` (9=중학교, 수=수학, 01=영역, 01=번호)
- 국어: `[9국01-01]`
- 영어: `[9영01-01]`
- 사회: `[9사01-01]`
- 과학: `[9과01-01]`
- 도덕: `[9도01-01]`
- 체육: `[9체01-01]`
- 음악: `[9음01-01]`
- 미술: `[9미01-01]`
- 정보: `[9정01-01]`
- 기술가정: `[9기가01-01]`

## 텍스트 줄바꿈

셀 내 줄바꿈은 `\n` 문자를 사용한다.
```json
"[수업]\n§ 발문식 수업\n§ 탐구활동\n[평가]\n§ 관찰평가"
```
